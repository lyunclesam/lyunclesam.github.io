Java里有哪些语法糖

Java作为一种与平台无关的高级语言，当然也含有语法糖，这些语法糖并不被虚拟机所支持，在编译成字节码阶段就自动转换成简单常用语法。
语法糖的大概有三个特性：

  1. 提高程序开发效率
  2. 程序性能不受影响
  3. 规避coder可能因手误而出现的错误

那Java有哪些语法糖呢？Java大概有如下6个语法糖：

  1. 泛型与类型擦除
  2. 自动装箱与拆箱
  3. 变长参数
  4. 增强for循环
  5. 内部类
  6. 枚举类
