Spring Boot的三种启动方式

Spring Boot有以下三种启动方式，最后一种我们可以很好的设置不同环境使用不同的配置文件。

1.IDE 运行Application这个类的main方法

2.在springboot的应用的根目录下运行mvn spring-boot:run

3.在springboot的应用的根目录下 mvn install 生成jar后运行
